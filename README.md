# 🪔 Diwali Sales Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on Diwali sales data to understand customer behavior and sales trends.

## 🛠️ Tools & Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

## 🔍 Key Insights
- Female customers contribute the most to total sales  
- The age group 26–35 is the most active and highest spending  
- Married women have the highest purchasing power  
- Top states contributing to sales are Uttar Pradesh, Maharashtra, and Karnataka  
- Customers from IT, Healthcare, and Aviation sectors spend the most  
- Top product categories are Clothing & Apparel and Food  

## 📊 Project Features
- Data Cleaning and Preprocessing  
- Handling Missing Values  
- Data Visualization using graphs  
- Customer Segmentation  
- Business Insights Extraction  

## 📁 Project Files
- Diwali_Sales_Analysis.ipynb → Python analysis notebook  
- cleaned_diwali_data.csv → Cleaned dataset  

## 🚀 Conclusion
This project helps in understanding customer purchasing behavior and provides useful insights for improving sales and marketing strategies.
